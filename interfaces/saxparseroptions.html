Optional xmlEnabled

xmlEnabled: undefined | false | true

Inherited from SaxParserDialectOptions.xmlEnabled

  • Defined in createSaxParser.ts:53

Determines whether XML rules should be applied during parsing.

If set to true then:

  • CDATA sections and processing instructions are parsed;
  • Self-closing tags are recognized;
  • Tag names are case-sensitive.

If set to false then:

  • CDATA sections and processing instructions are emitted as comments;
  • Self-closing tags are treated as start tags;
  • Tag names are case-insensitive.
default

false

  • Inherited from SaxParserCallbacks.onStartTag

    • Defined in createSaxParser.ts:110

      The name of the start tag.

    • attrs: ArrayLike<Attribute>

    • selfClosing: boolean
